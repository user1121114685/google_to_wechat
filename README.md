# google_to_wechat
## 将google voice 的新信息转发到微信
仅供参考学习golang  

在源码，或者编译的二进制文件**同级目录**中google_to_wechat_config.toml中配置好所需的信息，即可开始使用。  
  
如果md5 不通过 可以删除 go.sum 即可
